## Como buscar

### Filtros disponíveis

| Filtro | Descrição |
|--------|-----------|
| **Estado (UF)** | Filtre por unidade federativa |
| **Busca por texto** | Pesquise por instituição, cargo, banca ou palavra-chave |

### Status dos exames

Os exames são agrupados por status:

- **Abertos** — inscrições abertas
- **Em andamento** — período de provas
- **Previstos** — datas futuras
- **Encerrados** — processo finalizado

## Fluxo de uso

1. Acesse **Procurar Exames** no menu lateral.
2. Use os filtros para refinar a busca.
3. Clique em um exame para ver os **detalhes**.
4. Nos detalhes, você pode **selecionar** o exame como "Meu Exame" para acompanhar sua preparação.

## Particularidades

- Badge visual indica o estado (UF) selecionado
- Quando não há exames para os filtros aplicados, uma mensagem informativa é exibida
- A navegação para detalhes permite ver instituição, banca, datas, vagas, salário e mais
