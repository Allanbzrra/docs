Em **Assinatura > Planos** você visualiza os planos disponíveis, faz upgrade, downgrade ou inicia um trial.

<Frame>
  <img src="/images/gif-planos.gif" alt="Tela de Planos de Assinatura" />
</Frame>

## O que você encontra

- **Listagem de planos** — com features de cada um
- **Comparação** — lado a lado
- **Badges** — "Mais Popular", "Curadoria"
- **Trial grátis** — quando disponível

## Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Visualizar planos** | Ver todos os planos e features |
| **Escolher plano** | Selecionar o plano desejado |
| **Fazer upgrade** | Subir de plano (com confirmação) |
| **Solicitar downgrade** | Reduzir plano (com confirmação) |
| **Iniciar trial** | Usar período de teste grátis |
| **Selecionar método de pagamento** | Cartão para cobrança |
| **Configurar parcelas** | Até 12x no cartão |

## Particularidades

- **Cálculo de crédito proporcional** — ao fazer upgrade, o valor já pago é considerado
- **Validação de termos** — aceite dos termos antes de confirmar
