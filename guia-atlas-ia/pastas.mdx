As **Pastas** (também chamadas de *notebooks*) são onde você agrupa materiais de estudo por tema, disciplina ou projeto.

</Frame>

## O que você pode fazer

| Ação | Descrição |
|------|-----------|
| **Criar pasta** | Nova pasta para organizar materiais |
| **Listar pastas** | Ver ativas e arquivadas |
| **Buscar** | Pesquisar pastas por nome |
| **Abrir** | Entrar na pasta e ver conteúdo |
| **Arquivar** | Mover para arquivo (não exclui) |
| **Editar** | Alterar nome e descrição |
| **Excluir** | Remover a pasta |

## Fluxo recomendado

1. **Crie uma pasta** para cada disciplina ou tema (ex: "Direito Constitucional", "Matemática").
2. **Adicione materiais** dentro da pasta (links, PDFs, textos).
3. **Use as ferramentas** — podcasts, questões, flashcards — a partir dos materiais da pasta.

## Pastas arquivadas

Pastas arquivadas ficam em uma seção separada. Elas não aparecem na lista principal, mas continuam acessíveis. Use o arquivamento para organizar pastas que você não usa no momento.

<Tip>
Use nomes descritivos para encontrar rapidamente suas pastas na busca.
</Tip>
