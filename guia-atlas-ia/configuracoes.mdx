A página **Configurações** centraliza as preferências do usuário, opções de comportamento da IA e gerenciamento de assinatura.

<Frame>
  <img src="/images/gif-notebook-configuracoes.gif" alt="Tela de Configurações do Atlas IA" />
</Frame>

## O que você encontra

### Conta
- Editar perfil (nome, foto)
- Alterar senha
- Deletar conta

### Preferências de IA
- Modelo de chat padrão
- Modelo de embedding padrão
- Comportamento padrão nas respostas

### Assinatura
- Visualizar plano atual
- Ver limites e uso
- Link para upgrade

## Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Editar perfil** | Nome e informações da conta |
| **Alterar senha** | Segurança da conta |
| **Configurar IA** | Modelos e comportamentos padrão |
| **Salvar** | Persistir alterações |
| **Atualizar** | Recarregar configurações |

## Particularidades

- **Acessível a todos os usuários** — não requer ser admin
- **Persistência** — preferências salvas no servidor
