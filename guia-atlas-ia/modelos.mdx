A página **Modelos** permite configurar quais modelos de IA são usados para cada tipo de operação no Atlas IA — chat, embeddings, geração de áudio e transcrição.

<Warning>
Esta página é acessível apenas para **administradores**.
</Warning>

## Tipos de modelo

| Tipo | Usado em |
|------|----------|
| **Language (LLM)** | Chat, questões, resumos, redação, análises |
| **Embedding** | Busca vetorial, indexação de materiais |
| **Text-to-Speech** | Geração de podcasts e vídeos |
| **Speech-to-Text** | Transcrição de áudio |

## Status dos provedores

Cada provedor de IA (OpenAI, Anthropic, Google, etc.) tem um indicador de status:

- **Online** — conectado e funcionando
- **Offline** — sem conexão ou chave inválida

## Modelos padrão

Configure qual modelo é usado por padrão em cada tipo de operação. Isso afeta todo o sistema para todos os usuários.

## Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Ver status** | Verificar provedores ativos |
| **Configurar padrão** | Definir modelo padrão por tipo |
| **Atualizar** | Salvar configurações |

## Particularidades

- **Acesso restrito** — apenas administradores
- **Impacto global** — mudanças afetam todos os usuários
- **Múltiplos provedores** — OpenAI, Anthropic, Google, Ollama, etc.
