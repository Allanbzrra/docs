Ao abrir uma pasta, você acessa o **espaço de trabalho completo**: materiais organizados, notas criadas e um chat que responde com base no conteúdo da pasta.

## Layout em 3 colunas

| Coluna | Conteúdo |
|--------|----------|
| **Materiais** | Fontes adicionadas à pasta (links, PDFs, textos) |
| **Notas** | Anotações criadas manualmente ou geradas pela IA |
| **Chat** | Conversa com IA usando os materiais e notas como contexto |

## Materiais da pasta

### Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Adicionar material** | Link, arquivo ou texto |
| **Ver detalhes** | Abrir modal com conteúdo completo |
| **Modo de contexto** | Off / Insights / Full |
| **Remover** | Deletar material da pasta |

### Modos de contexto

- **Off** — o material não é enviado ao chat
- **Insights** — apenas os insights extraídos são usados
- **Full** — conteúdo completo é enviado como contexto

## Notas

### Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Criar nota** | Nova anotação manual |
| **Editar** | Alterar conteúdo da nota |
| **Modo de contexto** | Off / Insights / Full |
| **Excluir** | Remover nota |

## Chat com contexto

O chat usa os materiais e notas marcados como **Insights** ou **Full** para responder perguntas sobre o conteúdo da pasta.

### Fluxo de uso

1. Adicione materiais à pasta.
2. Configure o **modo de contexto** de cada material.
3. Faça perguntas no chat — a IA responde com base no contexto selecionado.
4. Crie notas a partir das respostas.

<Tip>
Use o modo **Full** para perguntas detalhadas sobre o conteúdo completo. Use **Insights** para respostas mais rápidas e objetivas.
</Tip>

## Gerenciamento da pasta

| Ação | Descrição |
|------|-----------|
| **Renomear** | Alterar nome da pasta |
| **Editar descrição** | Atualizar descrição |
| **Arquivar** | Mover para pastas arquivadas |
| **Excluir** | Remover pasta permanentemente |
