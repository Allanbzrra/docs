</Warning>

## O que você encontra

### Informações do sistema
- Versão atual do Atlas IA
- Status de conectividade
- Status dos modelos de IA configurados

### Rebuild de embeddings
Reprocessa os embeddings vetoriais de todos os materiais. Use quando:
- Buscas vetoriais não retornam resultados esperados
- Após migração de dados
- Após troca de modelo de embedding

### Utilitários
- Limpeza de caches
- Ferramentas de diagnóstico

## Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Ver informações** | Status e versão do sistema |
| **Rebuild embeddings** | Reprocessar incorporações vetoriais |
| **Acompanhar progresso** | Barra de progresso do rebuild |

## Particularidades

- **Acesso restrito** — apenas administradores
- **Rebuild com progresso** — barra visual acompanha o reprocessamento
- **Use com cuidado** — operações podem demorar dependendo do volume de dados
