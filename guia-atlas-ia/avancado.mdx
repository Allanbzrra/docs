A página **Avançado** reúne ferramentas de manutenção e utilitários para usuários avançados.

<Warning>
Esta página é acessível apenas para **administradores**.
</Warning>

## O que você encontra

- **Informações do sistema** — versão, status, diagnóstico
- **Rebuild de embeddings** — reprocessar incorporações vetoriais
- **Utilitários** — outras ferramentas de manutenção

## Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Ver informações** | Consultar status do sistema |
| **Rebuild embeddings** | Reprocessar embeddings de fontes |
| **Utilitários** | Acessar ferramentas de manutenção |

## Quando usar

- **Rebuild embeddings** — quando buscas vetoriais não retornam resultados esperados ou após migração de dados
- **Informações do sistema** — para diagnóstico de problemas

## Particularidades

- **Acesso restrito** — apenas admin
- **Ferramentas de manutenção** — use com cuidado
