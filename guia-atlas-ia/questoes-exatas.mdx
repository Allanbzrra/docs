**Questões de Exatas** extrai questões matemáticas de provas e documentos, permitindo resolver com editor LaTeX e receber correção detalhada passo a passo.

<Frame>
  <img src="/images/gif-notebook-questoes-exatas.gif" alt="Tela de Questões de Exatas do Atlas IA" />
</Frame>

## Como funciona

1. **Analise uma prova** — envie PDF ou documento com questões.
2. A IA **extrai automaticamente** as questões de exatas.
3. **Selecione uma questão** da lista.
4. **Resolva** usando o editor LaTeX integrado.
5. **Envie** para correção automática.
6. **Veja o feedback** com passo a passo da solução correta.

## Editor LaTeX

O editor suporta notação matemática completa:

- Frações, potências, raízes
- Integrais, somatórios, limites
- Matrizes e vetores
- Renderização em tempo real

## O que o feedback inclui

- Se correto: confirmação com mensagem de sucesso
- Se incorreto:
  - Passo a passo da solução correta
  - Identificação do erro cometido
  - Explicação dos conceitos envolvidos

## Metadados das questões

Cada questão extraída mostra:

| Campo | Descrição |
|-------|-----------|
| **Tópico** | Álgebra, Geometria, Cálculo, etc. |
| **Dificuldade** | Fácil, Médio, Difícil |
| **Enunciado** | Texto completo da questão |

## Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Analisar prova** | Enviar documento para extração |
| **Selecionar questão** | Escolher qual resolver |
| **Resolver** | Escrever no editor LaTeX |
| **Enviar resposta** | Submeter para correção |
| **Ver feedback** | Ler correção detalhada |
| **Excluir questão** | Remover da lista |

## Particularidades

- **Dois créditos consumidos** — `math_source` (análise) e `math_solution` (correção)
- **LaTeX renderizado** — visualização imediata das fórmulas
- **Extração inteligente** — ignora questões de outras disciplinas
