**Transformações** são prompts customizados que você pode aplicar em materiais para extrair, resumir ou reformatar o conteúdo de maneiras específicas.

## Como funciona

1. Acesse **Transformações** no menu Gerenciar.
2. Veja a lista de transformações disponíveis.
3. **Edite** o prompt padrão ou **crie** uma nova transformação.
4. **Teste** no playground com um texto de exemplo.
5. Aplique em materiais quando estiver satisfeito.

## O que você encontra

- **Lista de transformações** — todas as criadas com nome e descrição
- **Editor de prompt padrão** — prompt base do sistema
- **Playground** — ambiente de teste em tempo real

## Exemplos de transformações

- Extrair conceitos principais
- Resumir em bullet points
- Gerar perguntas sobre o conteúdo
- Reformatar em cronologia
- Criar glossário de termos

## Ações disponíveis

| Ação | Descrição |
|------|-----------|
| **Criar** | Nova transformação com prompt customizado |
| **Editar prompt padrão** | Alterar o prompt base do sistema |
| **Testar** | Usar o playground com texto de exemplo |
| **Aplicar** | Usar em uma fonte |

## Particularidades

- **Playground** — teste sem afetar dados reais
- **Prompts reutilizáveis** — aplique em múltiplas fontes
- **Acesso restrito** — apenas administradores
