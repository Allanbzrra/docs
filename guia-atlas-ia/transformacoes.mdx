**Transformações** são prompts customizados que você pode aplicar aos materiais para extrair, resumir ou reformatar o conteúdo de formas específicas.

<Warning>
Esta página é acessível apenas para **administradores**.
</Warning>

## O que você encontra

- **Listagem de transformações** — todas as transformações criadas
- **Editor de prompt padrão** — prompt usado por padrão
- **Criação** — novas transformações customizadas
- **Playground** — testar transformações antes de aplicar

## Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| **Criar** | Nova transformação com prompt customizado |
| **Editar** | Alterar prompt existente |
| **Testar** | Usar o playground para ver o resultado |
| **Aplicar** | Aplicar transformação em uma fonte |
| **Gerenciar** | Prompts padrão do sistema |

## Fluxo de uso

1. Acesse **Transformações**.
2. Crie ou edite uma transformação.
3. Defina o prompt (ex: "Extraia os principais conceitos", "Resuma em formato de bullet points").
4. Teste no **playground** com um texto de exemplo.
5. Aplique em uma fonte quando estiver satisfeito.

## Particularidades

- **Prompts reutilizáveis** — use em múltiplas fontes
- **Playground** — teste sem afetar dados reais
- **Acesso restrito** — apenas admin
